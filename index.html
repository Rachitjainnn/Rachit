Recent Projects

LinkConnect

LinkConnect is a full-stack web application that lets users create a personalized page to share all their important links. It supports Google login via NextAuth, background uploads to AWS S3, and an analytics dashboard built with Chart.js. Deployed on Vercel.

Tech Stack: Next.js | Node.js | Tailwind CSS | MongoDB | AWS S3 | NextAuth

Chill-o-Meter

Chill-o-Meter is a fun and intuitive AI-powered tool that helps you rephrase messages based on how “chill” or “formal” you want to sound. It uses a large language model (llama via groq) to generate personalized rewrites.

Tech Stack: Next.js | Node.js | Groq | Radix UI | Tailwind CSS

Dashboard

A clean, responsive Next.js dashboard for managing franchisee onboarding, tracking progress, viewing insights , handling leads, and answering pending questions—all in one place.

Tech Stack: Next.js | React | Tailwind CSS | Radix UI
